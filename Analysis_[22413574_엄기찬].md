## 1. Introduction

### 1) Summary

주식 투자에 입문하려는 초보 투자자들은 시황 파악에 상당한 어려움을 겪는다. 국내 지수(KOSPI, KOSDAQ 등)는 A 사이트, 미국 선물 지수는 B 사이트, 원자재 정보는 C 사이트, 뉴스 속보는 각 언론사 홈페이지를 개별적으로 방문해야 하는 불편함이 존재한다. 기존 금융 정보 플랫폼(네이버 증권, 인베스팅 닷컴 등)은 정보량이 방대하고 UI가 복잡하여 핵심 시황만 빠르게 파악하기 어렵다.

**StockEasy**는 이 문제를 해결하기 위해, 주요 시장 지표와 실시간 주요 속보를 단일 웹 페이지에서 제공하는 시황 요약 서비스다.

### 2) Features

- 국내 지수, 미국 선물, 원자재, 속보를 단일 페이지에서 제공
- 초보자도 쉽게 확인할 수 있는 직관적인 UI (색상으로 등락 구분)
- 자동 갱신 설정으로 별도 새로고침 없이 최신 데이터 유지

## 3. Domain Analysis

![alt text](image.png)

### 1) MainController

시스템의 진입점이자 중앙 제어 클래스이다. 사용자의 웹 페이지 접속 시 각 데이터 조회 클래스를 초기화하고 렌더링을 조율한다.

### 2) InquireDomesticIndex

사용자가 KOSPI, KOSDAQ, KOSPI200 등 국내 주요 지수를 조회할 때 사용하는 클래스이다. DataFetcher 클래스를 통해 외부 증권 API로부터 데이터를 수신하고, 현재가·등락폭·등락률을 색상과 함께 렌더링한다.

### 3) InquireUSFuturesIndex

사용자가 S&P500, 나스닥, 다우존스 선물 지수를 조회할 때 사용하는 클래스이다. 미국 선물 시장 운영 시간에 맞춰 데이터를 갱신하며, 장 마감 시에는 최종 종가를 표시한다.

### 4) InquireCommodityPrice

사용자가 금, WTI 원유, 브렌트유의 현재 시세를 조회할 때 사용하는 클래스이다. 미국 선물 시장 운영 시간과 동일하게 데이터가 갱신된다.

### 5) InquireRealtimeNews

사용자가 시황 속보를 조회할 때 사용하는 클래스이다. NewsCollector 클래스를 통해 RSS 피드를 수집하며, 발행 시각 기준으로 내림차순 정렬하여 최신 속보를 상단에 표시한다.

### 6) DataFetcher

외부 증권/원자재 API를 호출하고 응답 데이터를 파싱하는 클래스이다. InquireDomesticIndex, InquireUSFuturesIndex, InquireCommodityPrice 클래스에서 공통으로 사용한다.

### 7) NewsCollector

뉴스 RSS 피드를 수집·파싱하는 클래스이다. InquireRealtimeNews 클래스에 데이터를 제공한다.

### 8) AutoRefresher

설정된 갱신 주기(초 또는 분)에 따라 타이머를 실행하고, 주기 만료 시 DataFetcher와 NewsCollector를 호출하여 모든 데이터를 자동으로 갱신한다.

## 4. User Interface Prototype

### 1) 메인 페이지 레이아웃

![alt text](image-1.png)

### 2) 국내 지수 영역

DataFetcher가 호출한 API 결과를 기반으로 KOSPI, KOSDAQ, KOSPI200을 카드 형태로 출력한다. 각 카드에는 종목명, 현재가, 등락폭, 등락률이 포함된다.

### 3) 미국 선물 영역

InquireUSFuturesIndex 클래스가 수집한 S&P500·나스닥·다우존스 선물 데이터를 국내 지수와 동일한 카드 형태로 출력한다. 장 마감 시에는 "(장 마감)" 뱃지를 표시한다.

### 4) 원자재 영역

금, WTI, 브렌트유의 현재가와 변동률을 카드 형태로 출력한다.

### 5) 실시간 속보 영역

NewsCollector가 수집한 속보를 발행 시각 기준 내림차순으로 리스트 형태로 출력한다. 각 항목은 발행 시각과 헤드라인으로 구성된다.

## 5. Glossary

| 용어 | 설명 |
|------|------|
| KOSPI | Korea Composite Stock Price Index. 한국 유가증권시장의 대표 종합주가지수 |
| KOSDAQ | Korea Securities Dealers Automated Quotation. 한국의 기술주 중심 주식시장 지수 |
| 선물(Futures) | 미래의 특정 시점에 특정 자산을 미리 정한 가격으로 매매하기로 약정한 파생상품. 미국 선물 지수는 현지 장 개장 전 시장 방향성을 예측하는 지표로 활용됨 |
| WTI | West Texas Intermediate. 미국 텍사스산 경질 원유로, 글로벌 원유 가격의 기준 지표 중 하나 |
| 브렌트유 | 북해산 원유로, WTI와 함께 국제 원유 가격의 양대 기준 지표 |
| 시황 | 주식 시장의 전반적인 상황과 흐름 |
| RSS | Really Simple Syndication. 웹 콘텐츠를 표준화된 형식으로 배포하는 기술 |
| API | Application Programming Interface. 외부 서비스와 데이터를 주고받기 위한 인터페이스 |
| 등락폭 | 현재가와 전일 종가의 차이 |
| 등락률 | 등락폭을 전일 종가로 나눈 비율(%) |
