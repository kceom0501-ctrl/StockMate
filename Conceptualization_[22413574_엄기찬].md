**1.** Business Purpose

우리가 살고 있는 현재에는 돈을 벌 수 있는 수단이 굉장히 다양하다. 그 중에서도 일반인들도 흔하게 접할 수 있는 주식이 있는데, 대부분의 사람들은 주식에 대해 공부하지도 않고 큰 돈을 들이는 식으로 거의 도박을 하고있는 수준이다. 그리고 공부를 하기 위해 주식 투자에 입문하려고 하는 투자자들은 시황을 파악하기 위해 어떤 정보를 확인해야하는지, 심지어는 시황이라는 의미 자체를 모르는 사람들도 많다. 의미를 아는 투자자라고 해도 여러 사이트를 번갈아 방문해야 하는 불편함을 겪고 있다. 국내 지수, 미국 선물 지수, 원자재 가격, 실시간 주요 뉴스 속보 등의 정보가 각각 다른 플랫폼에 분산되어 있어 한 번에 파악하기 불편하다. 그래서 모든 주식시장의 정보를 한눈에 볼 수 있는 사이트의 필요성을 느꼈다.

StockEasy는 주식을 많이 접해보지 못한 초보 투자자들이 복잡한 금융 정보를 쉽고 빠르게 파악할 수 있도록, 주요 시장 지표와 실시간 주요 속보를 단일 웹 페이지에서 제공하는 시황 요약 서비스다. 이 웹사이트를 이용함으로써 초보를 비롯한 모든 주식 투자자들이 트레이딩을 하는데에 빠른 판단을 내릴 수 있도록 도와주는 것이 목표이다. 쉬운 접근성 및 실시간성이 이 웹사이트의 특징이다.

**4.** Concept of Operation

1) 국내 주요 지수 조회

| 항목 | 내용 |
|------|------|
| Purpose | 사용자가 KOSPI, KOSDAQ 등 국내 주요 지수를 실시간으로 파악하기 위함 |
| Approach | 사용자가 웹사이트에 접속하면 국내 증권 데이터를 보여준다. 현재가, 등락률, 등락폭은 색상(상승: 빨강, 하락: 파랑)으로 구분해 표시한다. |
| Dynamics | 웹사이트 접속 시 즉시 최신 데이터를 로드하며, 이후 일정 주기마다 갱신하는 것을 선택할 수 있도록 한다. |
| Goals | 국내 주요 지수를 한눈에 파악할 수 있는 실시간 지수 현황 기능을 구현한다. |

2) 미국 선물 지수 조회

| 항목 | 내용 |
|------|------|
| Purpose| 국내 장 개장 전 후로 미국 시장 분위기를 파악하기 위함 |
| Approach | 백엔드가 미국 선물 시장 데이터 API를 호출하여 S&P500, 나스닥, 다우존스 선물 지수를 수집한다. 수집된 데이터는 국내 지수와 동일한 형태로 표시된다.(미국은 상승이 파랑, 하락이 빨강이지만 초보 투자자는 혼돈이 생길 수 있기 때문에 통일) |
| Dynamics | 미국 선물 시장 운영 시간에 맞춰 데이터가 갱신되며, 장 마감 시에는 최종 종가를 표시한다. |
| Goals | 미국 선물 지수 현황 조회 기능을 구현한다. |

3) 원자재 시세 조회

| 항목 | 내용 |
|------|------|
| Purpose | 금, 원유 등 주요 원자재 가격을 파악하기 위함 |

| Dynamics | 미국 선물 시장 운영 시간과 동일하게 주기적으로 데이터가 갱신된다. |
| Goals | 주요 원자재 시세 조회 기능을 구현한다. |

4) 실시간 주요 속보 조회

| 항목 | 내용 |
|------|------|
| Purpose | 시황에 영향을 미칠 수 있는 주요 뉴스 속보를 실시간으로 제공하기 위함 |
| Approach | 단독뉴스 등 주요 뉴스 속보를 수집. 수집된 기사는 발행 시각 기준으로 정렬되어 최신 속보가 상단에 표시된다. |
| Dynamics | 실시간으로 뉴스 피드를 갱신하여 최신 속보가 신속하게 반영된다. |
| Goals | 주요 경제·금융 실시간 속보 조회 기능을 구현한다. |


**5.** Problem Statement

현황 및 문제점

주식 투자에 관심을 갖기 시작한 초보 투자자들은 시황 파악에 상당한 어려움을 겪는다. 국내 지수(KOSPI, KOSDAQ 등)는 A 사이트, 미국 선물 지수는 B 사이트, 원자재 정보는 C 사이트, 뉴스 속보는 각 언론사 홈페이지를 개별적으로 방문해야 하는 불편함이 존재한다. 기존 금융 정보 플랫폼(네이버 증권, 인베스팅 닷컴 등)은 정보량이 방대하고 UI가 복잡하여 초보 투자자가 핵심 시황만 빠르게 파악하기 어렵다.

해결 방향

StockEasy는 다음의 문제를 해결한다.
1. 국내 지수, 미국 선물, 원자재, 속보를 단일 페이지에서 제공하여 복수의 사이트를 방문할 필요가 없음
2. 초보자도 쉽게 확인할 수 있는 직관적인 UI
3. 자동 갱신 설정을 통해 별도의 새로고침 없이도 최신 데이터를 갱신할 수 있도록 함


**6.** Glossary

| 용어 | 설명 |
|------|------|
| KOSPI | Korea Composite Stock Price Index. 한국 유가증권시장의 대표 종합주가지수 |
| KOSDAQ | Korea Securities Dealers Automated Quotation. 한국의 기술주 중심 주식시장 지수 |
| 선물(Futures) | 미래의 특정 시점에 특정 자산을 미리 정한 가격으로 매매하기로 약정한 파생상품. 미국 선물 지수는 현지 장 개장 전 시장 방향성을 예측하는 지표로 활용됨 |
| WTI | West Texas Intermediate. 미국 텍사스산 경질 원유로, 글로벌 원유 가격의 기준 지표 중 하나 |
| 브렌트유 | 북해산 원유로, WTI와 함께 국제 원유 가격의 양대 기준 지표 |
| 시황 | 주식 시장의 전반적인 상황과 흐름을 의미 |
